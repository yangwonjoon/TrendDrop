# TrendDrop Kakao / BigKinds API Research

작성일: 2026-07-06

## 목적

한국형 트렌드 서비스의 보조 검색 수요 소스로서 카카오(다음) 계열 데이터와, "왜 뜨나"(D3) 뉴스 맥락 레이어 후보인 빅카인즈 Open API를 실사한다.

이 문서는 아래 질문에 답하는 것을 목표로 한다.

- 카카오데이터트렌드에 공식 API가 있는가
- 카카오 개발자 API 중 트렌드/콘텐츠 소스로 쓸 수 있는 것은 무엇인가
- 빅카인즈 Open API를 영리 스타트업이 쓸 수 있는가
- 빅카인즈가 막히면 뉴스 맥락 대안은 무엇인가

조사 방법: 공식 페이지(developers.kakao.com, bigkinds.or.kr, kakaocorp.com, 카카오 데브톡, 공공데이터포털) 직접 확인 + 웹검색 교차 검증. 일부 페이지(datatrend.kakao.com, lab.bigkinds.or.kr)는 조사 환경에서 접속 불가 — 미검증 항목에 명시.

## 한눈에 보는 결론

### 핵심 판단

- **카카오데이터트렌드는 API가 없다 — 공식 확인.** 카카오 데브톡에서 담당자가 "디벨로퍼스에서는 해당 API를 제공하지 않습니다"라고 답변(2025-02). 웹 전용이며, 자동화하려면 카카오 제휴 제안(with.kakao.com) 경로뿐
- 다음 실시간 이슈 검색어는 **2020-02 서비스 종료**, 대체 API 없음. 카카오톡/오픈채팅 트렌드도 외부 접근 경로 없음
- 카카오에서 쓸 수 있는 것은 **Daum 검색 API(블로그/카페/웹문서)를 "관련 콘텐츠" 소스**로 쓰는 것뿐 — 검색량 지표는 전혀 없음. 쿼터는 일 50,000건(종류별 30,000건)으로 충분
- **빅카인즈는 데이터가 이상적이지만 접근성 리스크가 높다.** 이슈랭킹·연관어·개체명 등 D3에 딱 맞는 기능이 있으나, 이용약관이 **사전 승낙 없는 영리 목적 이용을 금지**하고 자동화 수집·AI 학습 이용도 별도 계약 없이는 금지. 신청·요금·쿼터의 공개 기준이 없고 확인된 승인 사례는 대학·협약기관 중심
- 뉴스 맥락(D3)의 현실적 순서: **네이버 뉴스 API 주축 → 뉴스와이어 RSS/제휴 API 보강 → 빅카인즈는 KPF 직접 문의로 조건 확인**

### 판정표

| 소스 | 트렌드/맥락 가치 | 접근성 | 판정 |
|---|---|---|---|
| 카카오데이터트렌드 | 검색어 상대지수 (다음 기반) | ❌ API 없음 (공식) | 제외 (제휴 제안만 가능) |
| Daum 검색 API | 관련 콘텐츠 (블로그/카페) | ⭕ 무료, 일 5만 건 | 보조 채택 후보 |
| 카카오톡/오픈채팅 | — | ❌ 외부 미제공 | 제외 |
| 빅카인즈 Open API | 이슈랭킹·연관어·개체명 (D3 이상적) | 🔶 영리 이용은 사전 승낙/계약 필요 | 직접 문의 후 판단 |
| 뉴스와이어 RSS/API | 보도자료 (신제품 맥락) | ⭕ RSS 무료, API는 심사 | 보강 후보 |

## Part A — 카카오

## 1. 카카오데이터트렌드

공식 자료:

- https://datatrend.kakao.com
- 카카오 공식 보도자료: https://www.kakaocorp.com/page/detail/8806

무엇을 하는 서비스인가:

- 포털 다음(Daum) 통합검색어 기반 검색어 트렌드 조회 (2020년 오픈)
- 기간/기기/성별/연령/**지역** 필터, 검색어 최대 5개 비교, 결과 파일 다운로드 지원
- 수치는 **상대값**(구간 내 최대 검색량=100) — 절대량 아님 (제3자 리뷰로 확인, 공식 문구 미확인)

결정적 한계:

- **공식 API 없음** — 카카오 데브톡 담당자 공식 답변: "아쉽지만, 디벨로퍼스에서는 해당 API를 제공하지 않습니다" (https://devtalk.kakao.com/t/api/142265, 2025-02). 사업 제휴 필요 시 https://with.kakao.com 접수 안내
- 네이버 데이터랩과 동일하게 상대지수 구조인 데다, 지역 필터가 있다는 점 외에는 데이터랩 대비 우위가 뚜렷하지 않음

판정: **자동화 파이프라인에서 제외.** 수동 리서치 보조 또는 장기적으로 제휴 제안 검토 대상.

## 2. Daum 검색 API

공식 문서:

- https://developers.kakao.com/docs/latest/ko/daum-search/dev-guide
- 쿼터: https://developers.kakao.com/docs/latest/ko/getting-started/quota

무엇을 할 수 있나:

- 웹문서/동영상/이미지/블로그/책/카페 검색. `sort=recency` + `datetime` 필드로 최신 콘텐츠 수집 가능
- 응답: `meta`(total_count 등) + `documents`(title, url, datetime, blogname/cafename, thumbnail)

쿼터/비용: Daum 검색 전체 **일 50,000건**, 검색 종류별 **각 일 30,000건**, 플랫폼 전체 무료 쿼터 월 300만 건. 무료, 증설 신청 경로 있음.

한계:

- **검색량/트렌드 지표는 전혀 없음** — 콘텐츠 소스이지 수요 지표 소스가 아님
- 다음 실시간 이슈 검색어는 2020-02-20 종료 (https://www.segye.com/newsView/20191223512850), 인기 검색어 계열 API 부재

TrendDrop 활용: **D6(관련 콘텐츠 묶음)의 블로그/카페 축.** 네이버 검색 API와 함께 쓰면 커뮤니티성 콘텐츠 커버리지가 넓어짐. 쿼터도 넉넉해서 부담 없음.

## 3. 카카오톡 / 오픈채팅 데이터

- **공식 외부 접근 경로 없음.** 오픈채팅 API 공개 요청에 미제공 답변 반복 확인 (https://devtalk.kakao.com/t/api/14037 외). 오픈채팅 인기 키워드는 앱 내 노출 전용
- 외부 제공되는 카카오톡 API는 메시지 발송/공유/채널 계열뿐 → 트렌드 무관

## Part B — 빅카인즈 (BIG Kinds)

## 4. 빅카인즈 Open API

공식 자료:

- https://www.bigkinds.or.kr/v4/openApi/index.do
- 이용약관: https://www.kinds.or.kr/v2/account/agreement.do

무엇을 할 수 있나:

- 한국언론진흥재단(KPF) 운영. **참여 언론사 104개**, 1990년~현재 약 1억 건+ 기사
- Open API 기능(웹 교차 확인, 공식 매뉴얼은 승인 후 제공): 뉴스검색, 뉴스조회, **이슈랭킹**, 워드클라우드(**연관어**), 뉴스 타임라인, 인기검색어, 인용문 검색, 분야별 키워드, 키워드 추출 — **D3("왜 뜨나")에 이상적인 구성**

접근성 (핵심 리스크):

- 신청: 사용신청서 → 구글폼 제출 → 승인 → 매뉴얼+키 이메일 발급. 확인된 사례는 대학·협약기관 중심(사용 기간 1년 단위)
- **영리 기업 신청 가능 여부·요금·쿼터의 공개 기준 없음**
- 약관 제20조: 사전 승낙 없이 얻은 정보를 **영리 목적으로 이용하거나 제3자에게 이용하게 하는 것 금지**
- 약관 제21조: 로봇·스크래퍼 등 **자동화 수집 금지**, 콘텐츠의 **AI 학습·훈련 이용 일체 금지** (별도 계약/서면 동의 필요) — D3를 LLM 요약으로 구현할 계획이라면 이 조항이 직접 걸림
- FAQ: 분석 결과의 활용은 출처 표기로 가능하나 기사 본문 인용은 저작권 구매 필요
- 참고 경로: KPF '미디어 스타트업 지원사업'(뉴스 데이터 활용 지원, 기업당 최대 3억 사례) — 정식으로 뚫으려면 이 경로 병행 검토

판정: **데이터는 최적이나 무단 편입 불가.** KPF에 상업 이용 조건 직접 문의 + 지원사업 타진이 유일한 경로. 그전까지 MVP의 뉴스 맥락은 네이버 뉴스 API로.

## 5. 뉴스 맥락 대안 (빅카인즈가 막힐 경우)

1. **네이버 뉴스 검색 API** (기조사 → [naver-api-research.md](naver-api-research.md)) — 주축
2. **뉴스와이어** (https://www.newswire.co.kr/?md=A31) — 산업 156종 등 **203개 보도자료 RSS 무료** + 제휴사 전송 API(JSON, 심사 후). 뷰티/푸드/테크 신제품형 트렌드의 "왜 뜨는지" 보조 맥락으로 적합
3. **공공데이터포털** — KPF 제공 뉴스빅데이터 파일 데이터셋 74종 (주제 고정형·비실시간이라 부분적 활용만)
4. 연합뉴스 — 공개 API 미확인 (B2B 계약 방식 추정)

## TrendDrop에 대한 결론

- **카카오 축**: Daum 검색 API만 채택 후보(관련 콘텐츠 소스). 검색 수요 지표 소스로서의 카카오는 사실상 없음 — 네이버(데이터랩+검색광고)가 유일한 국내 검색 수요 축임이 재확인됨
- **뉴스 맥락(D3) 축**: MVP는 네이버 뉴스 API + 뉴스와이어 RSS 조합으로 시작. 빅카인즈는 KPF 문의 결과에 따라 2단계에서 편입 여부 결정
- 국내 소스 전반의 패턴: 데이터는 존재하지만 **공식 API가 없거나(카카오데이터트렌드, 데이터랩 인기검색어), 약관이 상업 이용을 막는(네이버 오픈API, 빅카인즈)** 구조 → 국내 축은 "제휴/문의로 뚫는 트랙"과 "당장 쓸 수 있는 트랙(검색광고 API, Daum/네이버 검색 API)"을 분리해서 진행해야 함
